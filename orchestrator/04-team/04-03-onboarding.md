# New Hire Onboarding

> MOC: [[00-meta/MOC - Team]]

## Pre-Day-1 Checklist

- [ ] Send welcome email with:
  - Company overview (link to [[01-company/01-01-overview]])
  - Access to this vault
  - Slack/Discord invite (or WhatsApp group)
  - Calendar invites for recurring meetings
- [ ] Set up accounts:
  - Admin dashboard access (if applicable)
  - Email address @tensee.my
  - GitHub org invite (for devs)
  - MongoDB Atlas read access (for ops)
- [ ] Prepare hardware (laptop, phone) if provided

## Day 1

- [ ] 1:1 with founder — mission, culture, expectations
- [ ] Walkthrough of this vault (30 min)
- [ ] Product demo — use the platform as a student would
- [ ] Admin dashboard walkthrough (if applicable)
- [ ] First task: shadow a real process (e.g., respond to 3 leads)

## Week 1

- [ ] Complete role-specific SOPs in [[08-sops/08-01-runbooks]]
- [ ] Shadow each core process once
- [ ] Write a "first impressions" note in `daily/`
- [ ] Weekly 1:1 with founder

## Month 1

- [ ] Own one process end-to-end
- [ ] Propose one improvement to a process or tool
- [ ] Present learnings at monthly review

## Role-Specific Tracks

### Student Placement Coordinator
- [ ] Learn lead qualification script
- [ ] Practice viewing scheduling (roleplay)
- [ ] Shadow 5 real student interactions
- [ ] Handle first solo booking

### Support Agent
- [ ] Read all chat transcripts from past 30 days
- [ ] Learn WhatsApp templates
- [ ] Practice escalation paths
- [ ] Handle first solo chat session

### Developer
- [ ] Set up local dev environment
- [ ] Read [[02-product/02-02-codebase-map]]
- [ ] Fix one labeled `good-first-issue` bug
- [ ] Deploy first feature to staging

### Marketer
- [ ] Review all current marketing assets
- [ ] Analyze last 30 days of lead sources
- [ ] Draft one campaign
- [ ] Present competitive analysis
